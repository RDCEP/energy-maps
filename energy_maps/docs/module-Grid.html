(inner) set_line_width(value, divisor)

Format electric grid line width for the calling grid class object. Create an exponential response curve between voltages and line widths to ensure that the lines that represent large voltages aren't too large visually.
Parameters:
Name Type Description
value Number the value to be scaled, bound to the line_width property of the corresponding `grid` object
divisor Number sets line inflection point to adjust the scale of the line width as `value` grows. Must be at least half the size of `value` to get a decent inflection point. Helps determine the mid point of the curve.
