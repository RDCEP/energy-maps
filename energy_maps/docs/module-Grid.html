(inner) grid :Object

A collection of electric grid classifications containing metadata that is used to filter out and render separate portions of the grid.
Type:
  • Object
Properties:
Name Type Description
identifier string an abbreviation of the class name in the data file
name string a nested property of `identifier`; cooresponds to class heading in data file
display string a nested property of `identifier`; display name for the legend
color string a nested property of `identifier`; rgba value used to set the color for the legend and the rendered viz
line_width string a nested property of `identifier`; used for mathematical operations that set the line width for the corresponding object
nominal Number voltage - a nested property of `identifier`; system voltage for the grid class (values are estimates)

(inner) set_line_width(value, divisor)

Format electric grid line width for the calling grid class object. Create an exponential response curve between voltages and line widths to ensure that the lines that represent large voltages aren't too large visually.
Parameters:
Name Type Description
value Number the value to be scaled, bound to the line_width property of the corresponding `grid` object
divisor Number sets line inflection point to adjust the scale of the line width as `value` grows. Must be at least half the size of `value` to get a decent inflection point. Helps determine the mid point of the curve.
